This is GUI based chat Application
Copy the given package into your project src folder
or import this netbeans project folder

1. First, Run (Shift + F6) the Server.java file.
2. Then, Run (Shift + F6) the ClientGUI.java file. 

Press Connect : To Connect to Server
				First, You have to login using your Username ,password and IP of the server
Press Signup : To Signup you have to enter your Username and password				
Press Send : To broadcast message to all available 
				Message can be sent by pressing enter
Press Disconnect : To Disconnect from Server

You also have to make a File.txt in the project folder which will contain your username and password of all those who have registered
